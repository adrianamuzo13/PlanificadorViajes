# Gestión del Proyecto
Uso de Trello para organizar el desarrollo del proyecto.
Enlace al tablero: https://trello.com/b/ZqbfLKjL/planificador-de-viajes

## Columnas del tablero
- **Backlog**: todas las tareas pendientes de planificar
- **Todo**: tareas listas para empezar
- **In Progress**: tareas en desarrollo activo
- **Review**: tareas terminadas pendientes de revisión
- **Done**: tareas completadas

Estoy usando Kanban porque las tareas pueden cambiar de prioridad en cualquier momento. Me permite visualizar el estado de cada funcionalidad.
